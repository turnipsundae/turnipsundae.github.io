Fit-j

This is my first web app project. Fit-j aims to help you discover and track your exercises. I couldn't find something that was simple and speedy on the internet, so I decided to create my own. Eventually I want the app to learn user preferences and make awesome workout suggestions. You should look forward to exercising!

Score Keeper App

This was my first Android app published on Google Play. I learned about xml, java, and other basic concepts around native Android apps. I learned to use stakeoverflow, blog tutorials, and other resources to implemented an engaging UI--it emulates Gmail's swipe to delete feature. I even tried to implement single tap and long tap in addition to the custom swipe animation, which was very challenging! Not everything worked according to plan. I should have removed the long tap which was causing the bugs, but kept it, convinced that more features meant better. The lesson? I learned that the best features are, firstly, working features.

Split Pea App

This was my second Android app published on Google Play. I learned about list adapters, launching activities, parceling data between activities, using share action provider, and much, much more! I tried to incorporate code from other projects, such as a calculator and the Tesseract ORC. However, I spent a week and couldn't implement either correctly, so moved on to using existing knowledge to build the essential functions. Having a peer review would have sped up my learning--next time I'll start or join a meet-up.
